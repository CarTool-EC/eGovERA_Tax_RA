Common Communication Network 2 (CCN 2) ()
Description: Common Communication Network 2 (CCN 2) is a pan-European telecommunications network that provides a secure and reliable platform for the exchange of information between the European Commission, EU Member States, and other participating parties. Additional information: CCN 2 is a high-speed network that supports the exchange of information for various applications and services related to the European Union's policies and activities. It is a key component of the EU's digital infrastructure, enabling the secure and efficient transmission of data across borders. The network is managed by the European Commission's Directorate-General for Taxation and Customs Union (DG TAXUD). It is used for a wide range of applications, including customs, taxation, and agricultural policy, among others. The network is designed to ensure the confidentiality, integrity, and availability of the data it transmits, using advanced security measures and protocols. Example: Examples of applications that use the CCN 2 network include the New Computerised Transit System (NCTS), which manages the movement of goods across EU borders, and the Excise Movement and Control System (EMCS), which monitors the movement of excise goods within the EU. Another example is the Integrated Administration and Control System (IACS), which supports the management of agricultural subsidies.
